## Aufgaben

### Teilbarkeit

Gib alle Zahlen von 1 bis 20 aus, die durch 3 teilbar sind. Schreibe dazu eine Funktion `teilbar(a, b)`, die die Teilbarkeit einer Zahl a durch eine andere Zahl b auswertet und entsprechend `True` oder `False` zurückgibt.


### Primzahlen

Schreibe eine Funktion:

```python
def ist_primzahl(n):
    ...
```


Die Funktion soll `True` oder `False` zurückgeben, je nachdem ob der übergebene Parameter `n` eine Primzahl ist oder nicht.

Der Benutzer soll dann in einer Schleife immer wieder eine Zahl eingeben können und das Programm teilt mit, ob sie prim ist oder nicht.

### Bubblesort

Informiert euch über die Funktionsweise des Bubblesort-Algorithmus (z.B. https://www.youtube.com/watch?v=Mj-payJDsdw) und implementiert ihn in Python als Funktion zum Sortieren von Listen.


### Sortierte Liste ohne Duplikate

Lies eine Liste von Zahlen vom Benutzer ein, oder gib eine statisch vor. Entferne alle Duplikate aus dieser Liste und sortiere anschließend die Liste. Für die Sortierung kannst du pythons `list.sort()`, oder auch den in der vorherigen Aufgabe erstellten Sortierfunktion.
